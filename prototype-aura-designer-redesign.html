Redesigned Aura Designer - Interactive HTML Prototype

Key changes from the current design:

  • Type Selector above tiles — Choose Icon, Square, or Bar FIRST, then the aura tiles show a preview of that type, making it clear what you'll be dragging
  • Visual type indicator on each tile — Each tile shows a small icon/square/bar preview badge so users see what they'll place
  • Right panel simplified — Instead of a long list of collapsible inactive effects, there's a single "+ Add Effect" button that opens a clean dropdown menu
  • Configured effects as cards — Only shows effects that are actually enabled, with expand/collapse for settings
  • Less intimidating — New users see a simple button instead of 8 disabled effect sections

Design Decisions & UX Improvements
  • Type Selector above tiles: The "Drag as: Icon | Square | Bar" selector tells users exactly what will happen when they drag a tile onto the frame. This eliminates the confusion of "what type will I get?"
  • Type indicator badge on each tile: A small icon/square/bar badge at the bottom of each tile previews what you'll be placing, reinforcing the type selector choice
  • "+ Add Effect" button: Instead of 8 collapsible sections (5 of which are disabled by default), there's one friendly button. Clicking it opens a dropdown where you choose which effect to add. This makes the feature feel approachable instead of overwhelming
  • Only configured effects shown: The right panel only displays effects that are actually enabled, as collapsible cards. This reduces clutter dramatically — a fresh aura shows just the "+ Add Effect" button
  • Dropdown with descriptions: Each effect in the dropdown has a one-line description, helping users understand what each effect does without having to enable it first
  • Active checkmarks in dropdown: The dropdown shows which effects are already enabled with a check mark, so users can see the full picture at a glance
  • "drag me" hint on tiles: Subtle hint text under each tile teaches users they can drag these onto the frame
