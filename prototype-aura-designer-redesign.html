Redesign Changes Summary

  • New Type Selector above tiles: Icon / Square / Bar toggle buttons sit above the aura tiles. Selecting a type changes the visual preview on each tile (icon thumbnail, colored square, or bar stripe). This makes it clear what you're about to place before dragging.
  • New "Add Effect" dropdown button: Replaces the old long list of 8 inactive collapsible sections. Users click "+ Add Effect" and pick from a categorized dropdown (Placed on Frame vs Frame-level). Only added effects appear as cards — no intimidating wall of disabled sections.
  • Changed Right panel simplified: Now shows only the selected aura's header, the Add Effect button, and compact effect cards. Each card is collapsible and shows its type badge + placement info. Much less visual noise.
  • Changed Effect cards are compact: Each active effect is a collapsible card with a colored type badge (Icon/Square/Bar/Border/etc.), placement info, and a delete button. Settings are grouped into clean sections with dividers.
  • Changed Drag workflow: Users select Icon/Square/Bar type first → see that visual on tiles → drag tile onto the frame. This matches the mental model of "choose what kind → place it".
  • Changed Active Effects strip: Unchanged concept, but now uses compact chips showing aura name + effect type + anchor, making it scannable at a glance.
